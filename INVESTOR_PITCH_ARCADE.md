- Big Tech (OpenAI, Google, Anthropic) controls AI models

- No alternative for privacy-conscious users/enterprises
- European companies forced to use US cloud providers

### 3. Crypto Token Confusion

  - **Securities** (investment products, heavy regulation)
  - **Governance tokens** (complex, unclear utility)
  - **Memecoins** (pure speculation, no real use)
- No clear model for **consumption-based utility tokens**

## Why Arcade Token?

### Traditional Token Models (Problems)

| Token Type | Problem |
|------------|---------|
| **Governance Token** | Security risk, unclear utility, complex |
| **Security Token** | Heavy regulation, KYC/AML, investor-only |
| **Memecoin** | Pure speculation, no real value |

### Arcade Token Model (Solution)

**Inspired by**: Arcade tokens, airline miles, Starbucks Rewards, Xbox Live Points

**Characteristics**:
- ‚úÖ **Consumption-based** - Buy, use, repeat (not hold)
- ‚úÖ **Stable price** - ~$0.001 per AJT
- ‚úÖ **Unlimited supply** - Minted on-demand
- ‚úÖ **Non-security** - No investment characteristics
- ‚úÖ **Regulatory clarity** - Similar to gift cards, loyalty points

**Precedents**:
- **Helium (HNT)** - Pay for IoT data
- **Filecoin (FIL)** - Pay for storage
- **Render (RNDR)** - Pay for GPU rendering

**a16z Endorsement**: In their November 2025 newsletter, a16z highlighted arcade tokens as an "undervalued but highly useful token class for onchain economies."
